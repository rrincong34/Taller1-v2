Taller JavaScript 1

Punto 6

Crear un algoritmo que almacene una lista de nombres en un arreglo y luego ordenarlos alfabéticamente. No se sabe la cantidad de datos que se ingresarán. Se ingresa una lista de nombres (la lista termina cuando se ingresa un espacio en blanco) si se repite un nombre, el sistema debe mostrar un mensaje indicando al usuario que no debe repetir nombres y continuar con la lista.
